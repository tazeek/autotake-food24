# autotake-food24

Automate process calculation of Intake24 data with the HEIFA scoring.
Purpose of this project is to reduce the time taken to calculate the recal
of a single survey.

## Steps involved

Uploading files
Processing files
Calculating the serves
Conversion to HEIFA scores
Writing to CSV file
